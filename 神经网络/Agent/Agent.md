## 概览

- **主题范围**：聚焦"Agent 在长任务中循环推理、累积状态时，上下文如何管理"这一 2026 年 Agent 工程的关键瓶颈。
- **核心洞察**：模型上下文窗口持续扩张（GPT-4.1 标称 1M / Llama 4 Scout 标称 10M），但**有效上下文远小于标称窗口**（NoLiMa 基准显示 11/13 LLM 在 32K 时跌到基线 50% 以下），Context Rot 与变更放大才是真正问题。
- **学习路径**：单篇笔记，通读即可。
