## 五、关键参数说明

### 精度说明

| 精度 | 位数 | 主要用途 |
|------|------|---------|
| FP64 | 64位 | 科学计算 HPC |
| FP32 | 32位 | 标准浮点 |
| TF32 | 19位 | Ampere+ AI 训练 |
| BF16 | 16位 | 深度学习训练推荐 |
| FP16 | 16位 | 传统 AI 训练 |
| FP8 | 8位 | Blackwell/Hopper 推理加速 |
| INT8 | 8位 | 量化推理 |

### 架构说明

| 架构 | 代表型号 | 特性 |
|------|---------|------|
| Blackwell | RTX 5090 | FP8 原生支持，DLSS 4，PCIe 5.0 |
| Hopper | H100, H200, H20 | Transformer Engine, FP8, MIG |
| Ada Lovelace | RTX 4090, RTX 6000 Ada, L40S | DLSS 3, 光追增强, FP8 |
| Ampere | RTX A6000, A100 | 第三代 Tensor Core |

### 选择建议

| 场景 | 推荐型号 |
|------|---------|
| 超大模型训练 (>70B) | H200 141GB |
| 大模型训练 (7-70B) | H100 80GB |
| 中国市场大模型 | H20 96GB / H800 80GB |
| 专业 AI 推理 | L40S / RTX 6000 Ada |
| 个人 AI 推理 | RTX 4090 / RTX 5090 |
| 专业渲染/视频 | RTX 6000 Ada / RTX 5000 Ada |
| 游戏 + AI | RTX 5090 / RTX 4090 |
