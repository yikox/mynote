# 预研

---

## 概览

- **主题范围**：聚焦 AIGC 时代的关键工程问题——扩散模型推理加速、量化、视频模型分析、图像编辑加速，以及代码诊断方法论。
- **学习路径**：
  - 扩散加速主线：[DiCache](./预研/DiCache扩散模型缓存加速.md) ↔ [SeaCache](./预研/SeaCache论文复现笔记.md)（同主题不同视角：自决定缓存 vs 频谱感知缓存）→ [SVDQuant](./预研/SVDQuant.md)（量化路线）→ [SDNQ](./预研/SDNQ项目分析报告.md)（开源项目分析）
  - 序列并行：[Triton-distributed](./预研/Triton-distributed序列并行Attention研究.md) → [神经网络/并行策略](../神经网络/并行策略/) 两条基础路线
  - 视频生成：[LTX-2.3](./预研/LTX-2.3模型总结.md)
  - 图像编辑：[RegionE](./预研/RegionE图像编辑.md) + [FPIE 泊松融合](./预研/图像编辑/FPIE泊松融合使用指南.md)
  - Bug 诊断：[代码诊断问题集](./预研/bugs/代码诊断问题集.md)
- **横向关联**：[神经网络](../神经网络/神经网络.md) 是上游基础知识（LLM 架构、并行策略、GPU 工具链），本目录是这些技术的"现在在做什么研究"前沿。

## 知识地图

### 扩散模型推理加速
六篇笔记形成完整版图：缓存加速（DiCache / SeaCache / SLA）+ 量化（SVDQuant / SDNQ）+ Triton-based 并行（Triton-distributed）。

- [DiCache扩散模型缓存加速.md](./预研/DiCache扩散模型缓存加速.md) — ICLR 2026，Online Probe Profiling + Dynamic Cache Trajectory Alignment，HunyuanVideo 2.34×
- [SeaCache论文复现笔记.md](./预研/SeaCache论文复现笔记.md) — CVPR 2026 Oral，频谱感知缓存（SEA-filtered 距离替代 raw L1 距离）
- [Triton-distributed序列并行Attention研究.md](./预研/Triton-distributed序列并行Attention研究.md) — 字节跳动 Seed，tile 级 compute-comm overlap，1.09-44.97×
- [Sparse-Linear-Attention技术分析.md](./预研/Sparse-Linear-Attention技术分析.md) — THU-ML，attention 权重解耦为"top-k 稀疏 + 低秩线性"两路
- [SVDQuant.md](./预研/SVDQuant.md) — MIT / Adobe / NVIDIA ICLR 2025 Spotlight，4-bit 量化，outlier 用低秩分量吸收
- [SDNQ项目分析报告.md](./预研/SDNQ项目分析报告.md) — Stable Diffusion 量化引擎项目阅读

### 视频生成模型
